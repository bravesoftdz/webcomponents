Overview

vkThread is a javascript plugin, which allows you to execute a piece of your code in a thread.

Plugin is built on HTML5 "Web Workers" technology. *

Almost any type of function can be opened in a thread with this plugin:

  • Regular functions
  • Object's methods
  • Functions which depend on library
  • Anonymous and Arrow functions
  • Lambda functions

One of the most useful features is vkhttp() function which let you to run AJAX in a thread. With this function you can open a thread, request data from a server, process it and send result to the UI.

vkThread supports both Promise and Callback development styles. However multithreading can be done only with Promise style. The reason I keep callback functionality is IE browser, which doesn't support native ES6 Promise object. So, if you must support IE family, use callback style.

Basic usage:

Install with bower


bower install vkthread

integrate plugun in your project


<script src="../vkthread/vkthread.min.js" type="text/javascript"></script>

create function

function foo(n, m) {
	return n + m;
}

execute this function in a thread 

/* ------- Promise style ------- */

var param = {
    fn: foo,     // <-- function to run in a thread;
    args: [2, 3] // <-- function's arguments;
};

vkthread.exec(param).then(
    function(data){
        console.log(data); // <-- prints 5;
    }
);

/* ------- Callback style ------- */

function display(data){ //<-- callback function;
    console.log(data);
}

var param = {
    fn: foo,     // <-- function to run in a thread;
    args: [2, 3], // <-- function's arguments;
    cb: display  // <-- callback function; can be anonymous
};

vkthread.exec(param);
